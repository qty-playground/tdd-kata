# TDD Kata 練習室

使用 Python 和 uv 從零開始練習 TDD。

## FizzBuzz 問題描述

實作一個函數 `fizzbuzz(n)`，根據以下規則回傳結果：

- 輸入 1-100 的數字
- 3 的倍數回傳 "Fizz"
- 5 的倍數回傳 "Buzz"  
- 15 的倍數回傳 "FizzBuzz"
- 其他數字回傳數字本身的字串

範例：
- fizzbuzz(1) → "1"
- fizzbuzz(3) → "Fizz"
- fizzbuzz(5) → "Buzz"
- fizzbuzz(15) → "FizzBuzz"
